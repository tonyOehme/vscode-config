# Setup Commands
## Mac
### clone repo
```
rm -rf ~/Library/Application\ Support/Code/User/
git clone https://github.com/tonyOehme/vscode-config.git ~/Library/Application\ Support/Code/User/
```
### just settings
```
curl https://raw.githubusercontent.com/tonyOehme/vscode-config/main/keybindings.json?token=GHSAT0AAAAAACPQZNFUDNNG3NILNVOZ3QLWZP2F6KA > ~/Library/Application Support/Code/User/keybindings.json
curl https://raw.githubusercontent.com/tonyOehme/vscode-config/main/settings.json?token=GHSAT0AAAAAACPQZNFVIRJTULPIBA43FO4SZP2GDRQ > ~/Library/Application Support/Code/User/settings.json
```
